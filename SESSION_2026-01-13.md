### 3. Bug Fix - forge.fanz.website 502 Error

**Problem:** Nginx was proxying to wrong port (4006 instead of 3012)

**Solution:**
```bash
sed -i 's/127.0.0.1:4006/127.0.0.1:3012/g' /etc/nginx/sites-available/forge.fanz.website
pm2 delete forge && pm2 start ecosystem.config.cjs
nginx -t && systemctl reload nginx
```

### 4. Server C Cleanup

```bash
systemctl reload nginx
```
