## <a name="setup"></a>🎬 Setting Up Your First Stream

### Before You Go Live

**1. Plan Your Content**

Decide what you'll stream:
- Q&A session with fans
- Behind-the-scenes content
- Workout session
- Get ready with me
- Gaming
- Cooking/tutorial
- Just chatting
- Special announcement
- Exclusive performance

**2. Promote Your Stream**

Let fans know you're going live:
- Post announcement 24 hours before
- Send mass message to subscribers
- Post teaser on your feed
- Story countdown (if available)
- Pin announcement to profile

**Example announcement:**
```
🔴 GOING LIVE TOMORROW! 🔴

Join me for an exclusive Q&A session!
📅 Tuesday, 8 PM EST
💬 Ask me anything!
🎁 Exclusive content reveals
💰 Tips = extra special shout-outs!

Subscribers only 🔒
See you there! ❤️
```

**3. Prepare Your Space**

**Lighting:**
- ✅ Face the light source (window or lamp)
- ✅ Avoid backlighting (light behind you)
- ✅ Ring light recommended for best results
- ✅ Natural light works great (daytime)

**Background:**
- ✅ Clean, uncluttered space
- ✅ Neutral or branded background
- ✅ Remove personal items (privacy)
- ✅ Consider virtual backgrounds

**Audio:**
- ✅ Quiet environment
- ✅ Close windows (reduce noise)
- ✅ Test microphone beforehand
- ✅ External mic recommended (but not required)

**Camera:**
- ✅ Eye level (not looking up or down)
- ✅ Centered in frame
- ✅ Clean camera lens
- ✅ Check preview before going live

## <a name="going-live"></a>🔴 Going Live

### Pre-Stream Checklist

Before clicking "Start Stream":

✅ **Technical:**
- Camera working and positioned correctly
- Microphone tested
- Internet connection stable (5+ Mbps upload)
- Good lighting
- Battery charged (if mobile)

✅ **Content:**
- Stream title and description set
- Privacy level selected
- Tip goals set (optional)
- Talking points prepared
- Promotional materials ready (if doing giveaway, etc.)

✅ **Appearance:**
- Hair/makeup done (if applicable)
- Outfit chosen
- Background clean
- Looking at preview - everything good?

### Starting Your Stream

**Desktop:**
1. Go to Dashboard → Go Live
2. Fill in stream details
3. Click "Preview" to check camera/mic
4. Click "Start Stream"
5. You're live! 🔴

**Mobile:**
1. Tap camera icon in app
2. Select "Go Live"
3. Fill in details
4. Preview camera
5. Tap "Start Broadcasting"

**What happens when you go live:**
- 🔔 Subscribers get notification (if enabled)
- 🔴 Red "LIVE" badge appears on your profile
- 💬 Chat opens for viewers
- 📊 View count shows in real-time
- 💰 Tips can start coming in

### Your First 60 Seconds

**The stream opening is critical!**

**Do this in first minute:**
1. **Greet viewers warmly**
   - "Hey everyone! Thanks for joining!"
   - "Welcome to the stream! So glad you're here!"

2. **Introduce what you're doing**
   - "Today we're doing a Q&A session!"
   - "I'm going to show you behind-the-scenes of my photoshoot!"

3. **Encourage engagement**
   - "Drop a hi in the chat!"
   - "Let me know where you're watching from!"

4. **Mention tips (casually)**
   - "Tips are appreciated and help me keep doing these!"
   - "Top tipper today gets a special shout-out!"

**Example opening:**
```
"Hey everyone! Welcome to the stream! I'm so excited to
hang out with you all tonight. We're going to do a Q&A,

peek of something special...
```

**Fill dead air:**
- Talk about what you're doing
- Share stories

- Describe your surroundings

**Moderate your chat:**
- Delete inappropriate messages
- Ban abusive users
- Set chat rules upfront
- Focus on positive interactions

**While streaming, you can:**

- 📊 **View count** - See how many watching
- 💬 **Chat** - Read and respond to messages
- 💰 **Tip tracker** - See tips in real-time

- 📷 **Flip camera** - Front/back camera (mobile)
- 🎛️ **Filters** - Apply beauty filters (optional)

**Pausing Your Stream:**
- Use sparingly (bathroom, technical issue)
- Viewers see "Stream Paused" screen
- Chat remains active

- Chrome 90+

- iOS 14+ or Android 10+

## <a name="troubleshooting"></a>🔧 Troubleshooting

### Stream Won't Start

**Causes & Solutions:**

❌ **Camera/microphone not detected**
→ Check browser permissions
→ Allow camera/mic access
→ Refresh page and try again

❌ **Internet too slow**
→ Run speed test (need 5+ Mbps upload)
→ Close other apps using bandwidth
→ Switch to wired connection

❌ **Browser not supported**
→ Update to latest version
→ Try different browser (Chrome recommended)

❌ **Account not verified**
→ Complete ID verification first
→ Settings → Verification

### Poor Stream Quality

**Blurry or pixelated:**
→ Increase lighting
→ Check internet speed
→ Clean camera lens
→ Reduce motion (move slower)

**Laggy or freezing:**
→ Lower stream quality setting
→ Close background apps
→ Use wired internet
→ Reduce upload resolution

**Audio issues:**
→ Check microphone connection
→ Adjust audio levels
→ Reduce background noise
→ Use external mic

### Stream Keeps Disconnecting

**Solutions:**
→ Check internet stability
→ Switch from WiFi to wired
→ Close bandwidth-heavy apps
→ Contact ISP if persistent

**Temporary disconnection:**
- Stream auto-reconnects within 30 seconds
- Viewers see "Reconnecting..." message
- Chat stays active
- Resume from where you left off

### Viewers Can't See/Hear You

**Video frozen:**
→ Tell viewers to refresh
→ Check your camera connection
→ Pause and resume stream

**No audio:**
→ Check if you're muted
→ Verify mic permissions
→ Test with different mic

**Chat not working:**
→ Refresh page
→ Check if chat is enabled in settings
→ Clear browser cache

## <a name="faq"></a>❓ FAQ

**Q: How long should my streams be?**
A: 30-45 minutes is optimal. Short enough to keep attention, long enough to build connection and earn tips.

**Q: How much can I earn from streaming?**
A: Varies widely. Beginner: $20-$100/stream. Established: $200-$500/stream. Top creators: $1,000+ per stream.
