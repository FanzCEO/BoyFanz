## 🎯 Step-by-Step Verification Process

### Step 1: Start Verification

**Navigate to verification:**
1. Click your **Profile** picture (top right)
2. Select **Settings**
3. Click **Verification** tab
4. Click **"Get Verified"** button

**Or use quick link:**
- Dashboard → Quick Actions → **"Get Verified"**

---

### Step 2: Upload Government ID

**Taking ID photo:**

**✅ DO:**
- Use good lighting (natural light best)
- Lay ID flat on dark background
- Ensure all 4 corners visible
- Keep camera steady
- High resolution (no blur)
- All text clearly readable

**❌ DON'T:**
- Use flash (creates glare)
- Crop or edit the photo
- Cover any information
- Use photocopies or screenshots
- Angle or tilt the ID

**Upload process:**
1. Click **"Upload ID"**
2. Select front of ID from device
3. Review preview (check clarity)
4. If back required, upload back side
5. Click **"Next"**

**Tips:**
- Take photo in daylight if possible
- Remove ID from wallet/holder
- Use dark surface (not white)
- Retake if blurry or unclear

---

### Step 3: Take Live Selfie

**Why live selfie?**
Confirms you're the person on the ID (prevents fraud).

**How to take perfect selfie:**

**Setup:**
- Find well-lit area (face the light)
- Remove hats, sunglasses
- Hair away from face
- Neutral expression
- Look directly at camera

**Taking the photo:**
1. Click **"Take Selfie"**
2. Allow camera access
3. Position face in frame guide
4. Hold still
5. Click capture or auto-capture
6. Review and retake if needed

**Common issues:**
- Too dark: Move to brighter area
- Blurry: Hold phone steady
- Face not visible: Remove obstructions
- Doesn't match ID: Ensure good lighting

---

- Bio verification (if celebrity)

## 🐛 Troubleshooting

### "ID Rejected - Not Clear"

**Problem:** Photo too blurry or dark

**Solution:**
1. Retake in better lighting
2. Use rear camera (better quality)
3. Steady phone on flat surface
4. Ensure all text readable
5. Resubmit new photo

---

### "Selfie Doesn't Match ID"

**Problem:** Can't verify it's the same person

**Solution:**
1. Improve lighting (face the light)
2. Remove accessories (hat, glasses)
3. Match ID photo expression
4. Ensure face fully visible
5. Retake and resubmit

---

### "Information Mismatch"

**Problem:** Details don't match ID

**Solution:**
1. Check name spelling exactly
2. Verify date of birth format
3. Confirm all fields match ID
4. No typos or extra spaces
5. Resubmit with corrections

---

### "ID Expired"

**Problem:** Document is past expiration

**Solution:**
- Renew your ID first
- Use different valid ID
- Contact support if urgent
- Cannot verify with expired ID

---

### "Already Verified Account Exists"

**Problem:** ID used for another account

**Solution:**
1. Check if you have another account
2. Contact support to merge accounts
3. Verify this is your only account
4. Provide explanation to support

---

### "Verification Taking Too Long"

**Problem:** Over 24 hours, no response

**Solution:**
1. Check spam folder for emails
2. View status in dashboard
3. Contact support after 48 hours
4. Provide submission timestamp
5. Request status update
