### Create Custom Templates

1. **Go to:** Settings > Message Templates
2. **Click "Create Template"**
3. **Fill in details:**
   ```
   Name: Thank You VIP
   Category: Appreciation
   Template:
   "Hey {fan_name}! 💕
   ```

### Template Variables
- `{fan_name}` - Fan's username
- `{creator_name}` - Your name
- `{date}` - Current date
- `{time}` - Current time
- `{days_subscribed}` - Days since they subscribed
- `{total_spent}` - Their lifetime spending

## 🆘 Troubleshooting

### Messages Not Sending
**Solutions:**
1. Check internet connection
2. Verify recipient hasn't blocked you
3. Try refreshing the app
4. Clear app cache
5. Contact support if issue persists

### Can't Find a Message
**Use advanced search:**
1. Tap search icon 🔍
2. Try different keywords
3. Use filters (date, fan, status)
4. Check archived messages
5. Messages are never deleted, just hidden

### Fan Says They Didn't Receive Message
**Check delivery status:**
- ✓ Sent (delivered to server)
- ✓✓ Delivered (received by fan)
- 👁️ Read (fan opened message)

**If not delivered:**
- Fan may have connection issues
- Check if they blocked you
- Resend the message
